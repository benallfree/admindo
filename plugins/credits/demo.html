ES Module Integration

This plugin is now an ES module. You can import it like this:

import creditsPlugin from './plugins/credits/index.js' // or const creditsModule = await import('./plugins/credits/index.js')

The components use Shadow DOM for style encapsulation and work in all modern browsers that support Custom Elements v1 and ES modules.
